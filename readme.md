## 🌟 核心模块与功能

### 🎯 模块 1：批量出图章工具 (主打)
专为不同尺寸图纸混合的 PDF 设计，支持多图章并行加盖与精准定位。
* **智能自适应：** 导入印章（支持透明 PNG），系统自动计算原始宽高比。
* **物理尺寸分段：** 自动识别原 PDF 的页面物理尺寸（如 A3, A4）并分组，只需为每种尺寸设置一次印章位置即可全篇应用。
* **终极自由微调：** 提供可视化大图预览，支持对任意单页的图章进行位置拖拽、独立旋转、快速复制。
* **配置复用：** 排版配置可导出为 JSON。下次处理同尺寸图纸时一键导入，智能对齐历史坐标。
* **🛡️ 防篡改数字签名锁：** 独家支持导入 `.pfx` 证书。加盖图章的同时生成合法数字签名，锁定图章区域，防止被任何 PDF 编辑器（含 Acrobat）二次篡改或移动，确保文档法律效力。

### 📖 模块 2：OCR 智能提取目录
内置先进的 **PaddleOCR** 引擎，纯本地离线运行，安全且极速。
* **指哪打哪：** 在预览图上直观拖拽彩色选框，设定需提取的区域。
* **动态字段：** 支持无限添加目标字段（如：图纸名称、图号、版本号、设计人等）。
* **智能配置继承：** 提取框配置可保存，下次导入新图纸时自动匹配物理宽高对齐。
* **深度联动：** 提取出的表格数据不仅支持导出 Excel，更支持 **直接作为书签写入 PDF** 或 **按提取名称将文档拆分为单页**。

### ✂️ 模块 3：超级裁剪拼接大师
* 支持将 JPG 图片和 PDF 文件混装拖入处理。
* 极度灵活的切线交互：自由添加横/竖切割线，鼠标拖拽移动，右键删除。
* **暗化剔除：** 点击任意被切割的网格使其变暗，导出时将完美剔除该区域（轻松去除冗余白边或废弃印章）。

### 🗜️ 模块 4：全能文件压缩
* **PDF 强力压缩：** 底层内置 **Ghostscript** 引擎（用户无需额外安装）。提供 72(网络传输)、150(平衡)、300(高清打印) 三挡 DPI 无损/有损压缩方案。
* **图片批量压缩：** 支持 JPG/PNG/WEBP 批量转换，可自定义分辨率或按长边像素进行智能缩放。

### 🛠️ 模块 5：综合工具箱
集成了日常高频的文档处理痛点工具：
* 极速多图/多 PDF 顺序合并。
* PDF 逐页拆分。
* **高级按书签拆分：** 可按“书签名称”或“序号+书签”拆分，甚至支持按页码段自定义追加前后缀。
* **PDF 转纯图片型 PDF：** 专治字体缺失、乱码的顽固 PDF，将其底层栅格化重组为绝对兼容的只读 PDF。
* PDF 批量导出为高清 JPG/PNG（支持 PNG 透明通道）。

### 📐 模块 6 (全新)：线稿图片转 DXF 引擎
专为设计师与工程师打造，基于 OpenCV 高级边缘检测与多边形拟合算法，将手绘线稿、图纸扫描件极速矢量化为 CAD 标准的 `.dxf` 格式。
* **毫秒级商业预览体验：** 告别盲目瞎转。提供可视化的渲染视窗，拖动滑块即可实时（零延迟）预览红色 CAD 线条的提取结果。
* **智能去噪与平滑降维：** 独家提供“节点平滑度”算法滑块。可在保留图纸特征的前提下，大幅减少 DXF 文件体积并消除锯齿。
* **极简模式：** 智能应对“白底黑线”与“黑底白线”图纸，纯本地批处理生成，断网可用。

## 💻 技术栈

* **GUI 框架:** `PyQt5`
* **PDF 核心处理:** `PyMuPDF` (fitz), `PyPDF2`, `pypdfium2`
* **OCR 引擎:** `paddlepaddle`, `paddleocr`
* **图形图像与矢量化:** `opencv-python`, `ezdxf`, `Pillow`
* **底层压缩渲染:** `Ghostscript` (已内置便携版)
* **防伪与数字签名:** `pyhanko`, `cryptography`

## 🛠️ 源码运行指南

本项目基于纯 Python 开发，如果您希望在本地运行源码，请按照以下步骤操作：

**1. 克隆仓库**
​```bash
git clone https://github.com/XYD-Studio/XYD-PDF-Tools.git
cd 您的仓库名
```

**2. 创建虚拟环境 (强烈建议)**
由于包含 PaddleOCR 等大型深度学习库，建议使用虚拟环境隔离。
```bash
python -m venv .venv
# Windows 激活虚拟环境:
.venv\Scripts\activate
# macOS/Linux 激活虚拟环境:
source .venv/bin/activate
```

**3. 安装依赖**
```bash
pip install -r requirements.txt
```

**4. 启动程序**
```bash
python main.py
```
